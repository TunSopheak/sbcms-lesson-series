Setup Steps

1
Rename Figma File
ចូល Figma → Open File ចេញពី Part 3/4 → Double-click File Name → Rename: "SBCMS_Wireframes"
2
Add Color Styles
Draw Rectangle → Fill: #3B2007 → Right Panel → Fill → Click "⊕ Style" → Save as "Roast/Primary" ។ Repeat for ​Espresso, Caramel, Cream, Milk ។
3
Add Text Styles
Select Text → Right Panel → Type → "⊕ Style" → Save as "H1/Display", "Body/Regular", "Label/Mono" ។
4
Create 5 Screen Pages
Left Panel → Pages → "+" × 5 → Rename ដូច Structure ខាងលើ ។ ការ Organize Pages = Professional Workflow ។
5
Frame Setup Per Screen
ជ្រើស Page → ចុច F → Right Panel → Frame Preset → "Desktop" → W:1440, H:1024 → Fill: #FDF6EC (Cream) ។
☕
Figma Color Styles = SBCMS Design System
ក្រោយ Save Color Style → Elements ទាំងអស់ប្រើ Style Name ។ ប្ដូរ Color ១ → UI ទាំងអស់ Update ។ Professional Figma Files Always ប្រើ Styles ។

Dev Mode → C# WinForms Translation 
// Figma Button Values → C# WinForms Properties
button1.BackColor = ColorTranslator.FromHtml("#3B2007");
button1.ForeColor = ColorTranslator.FromHtml("#FDF6EC");
button1.Size      = new Size(160, 44);
button1.Font      = new Font("Segoe UI", 9.75f, FontStyle.Bold);
button1.FlatStyle = FlatStyle.Flat;
button1.FlatAppearance.BorderSize = 0;

// Figma Input Field → C# TextBox
textBox1.BackColor    = Color.White;
textBox1.BorderStyle  = BorderStyle.FixedSingle;
textBox1.Font         = new Font("Segoe UI", 9.75f);
textBox1.Size         = new Size(300, 40);
textBox1.Padding      = new Padding(9, 0, 14, 0);
💡
Figma → WinForms Mapping Tips
Figma "px" → WinForms "Size(w,h)" ។ Figma "Fill Color (Hex)" → ColorTranslator.FromHtml("#hex") ។ Figma "Border Radius" → WinForms Custom Panel/Button ដោយ GraphicsPath ។ Figma "Auto Layout Gap" → WinForms FlowLayoutPanel.Spacing ។
