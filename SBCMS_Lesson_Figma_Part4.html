Section 01

Component ជាអ្វី?

Component ជា Reusable UI Element ដែល Designer បង្កើត ១ ដង — ហើយ ប្រើម្ដងទៀត ពេញ Design File ។ ប្ដូរ Master Component → Instances ទាំងអស់ប្ដូរ Auto ។

⬡
ហេតុអ្វីប្រើ Components?
ដោយគ្មាន Component: Button ១ ប្ដូរ Color → ត្រូវ Update ១០០ Buttons ដោយដៃ ។
ដោយប្រើ Component: ប្ដូរ Master Component → Instances ទាំង ១០០ ប្ដូរ Auto ១ click ។
Component = Design System Foundation ។

Component vs Instance

MASTER COMPONENT
⬡ Btn/Primary
Confirm Order
▸ Master = Figma Store ។
▸ លេចក្នុង Assets Panel ។
▸ ប្ដូរ → ប៉ះពាល់ Instances ។
INSTANCE
⬡ Btn/Primary (copy)
Login
Save
▸ Instance = Copy ពី Master ។
▸ Override Text/Icon បាន ។
▸ Style Inherit ពី Master ។
☕
Analogy — ដូច Mold
Master Component = Mold (ស្ទាំង) ។ Instance = Product ពី Mold ។ ប្ដូរ Mold → Products ទាំងអស់ប្ដូរ ។ ប៉ុន្តែ Product ម្នាក់ អាច Override Color/Text ដោយមិនប៉ះពាល់ Mold ។
Section 02

Create Component — របៀបបង្កើត

Figma Component បង្កើតយ៉ាងងាយ — Select Element → ចុច Ctrl + Alt + K ឬ Right-Click → "Create Component" ។

Step-by-Step — Create Button Component

1
Draw Button Shape
ចុច R → Draw Rectangle W=160, H=44 → Fill: #3B2007 → Corner: 8 ។
2
Add Text Label
ចុច T → វាយ "Confirm Order" → Font: Kantumruy Pro 13px Bold → Color: #FDF6EC → Center ក្នុង Rectangle ។
3
Group ហើយ Select ទាំងពីរ
Shift+Click ជ្រើស Rectangle + Text → ឬ Ctrl+A Select All ។
4
Create Component
Ctrl + Alt + K ឬ Right-Click → "Create Component" → Component Icon (⬡) លេចក្នុង Layers Panel ។
5
Rename Component
Double-click Name ក្នុង Layers → វាយ "Btn/Primary" → Enter ។ Slash "/" = Folder Group ក្នុង Assets Panel ។
6
ប្រើ Assets Panel
Left Panel → Assets Tab → Local Components → "Btn" Folder → Drag "Btn/Primary" ទៅ Canvas = Instance ។

Section 03

Instances & Overrides — ការប្រើម្ដងទៀត

Instance ជា Copy ពី Master Component ។ Instance អាច Override (ផ្លាស់ប្ដូរ) Text, Icon, Color ដោយ មិន Break Link ជាមួយ Master ។

Override អ្វីបាន?

✅ Override បាន
  • ✓ Text Content (Label)
  • ✓ Icon (ប្ដូរ Emoji/SVG)
  • ✓ Image Fill
  • ✓ Visibility (Show/Hide Layer)
❌ Override មិនបាន (ដូច Master)
  • ✗ Layout Structure
  • ✗ Auto Layout Direction
  • ✗ Corner Radius (Global)
  • ✗ Fill Color (Default)
🔄
Swap Component
Instance ១ អាច Swap ជា Component ផ្សេង ។ Right-Click Instance → "Swap Component" → ជ្រើស Component ថ្មី ។ Text Override រក្សា ។ Style ប្ដូរ ។ ប្រើ ច្រើនក្នុង Variant System ។

Detach Instance

ប្រសិនចង់ Edit Instance ដោយ Free — Right-Click → Detach Instance (Ctrl + Alt + B) ។ Instance ក្លាយជា Group ធម្មតា — Link ជាមួយ Master ដាច់ស្ទើរ! ប្រើ Detach ក្នុង Cases ពិសេសប៉ុណ្ណោះ ។

Section 05

Auto Layout — ដំឡើង Layout Auto

Auto Layout ជា Feature ដ៏សំខាន់បំផុតក្នុង Figma ។ វាធ្វើឱ្យ Frame Resize Automatically នៅពេល Content ផ្លាស់ប្ដូរ — ដូច Flexbox ក្នុង CSS ។

⚡
Auto Layout = Flexbox in Design
CSS Flexbox: display:flex; gap:8px; padding:10px 20px;
Figma Auto Layout: Direction + Gap + Padding → ដូចគ្នាបេះបិទ ។
អ្នក Dev អាចប្ដូរ Figma Design ទៅ CSS បានដោយ ងាយ ។

Practice — Build SBCMS Button System

Exercise នេះ Build SBCMS Button Component ជា Variant System ក្នុង Figma — Default, Accent, Outline, Disabled ។

Exercise 02 — Button Component + Auto Layout

Task: Create Btn/Primary with 4 Variants + Auto Layout

01 បើ Figma File ចេញពី Part 3 → បន្ថែម Page ថ្មី → Rename "_Components"
02 ចុច R → Draw Rectangle W=160, H=44 → Fill: #3B2007 → Corner: 8
03 ចុច T → វាយ "Confirm" → Kantumruy Pro · 13px · 600 · Color: #FDF6EC → Center ។
04 Select Rectangle + Text → Add Auto Layout (Shift + A) → Direction: Horizontal · Gap: 8 · Padding: 10px 20px
05 Select Frame → Ctrl + Alt + K → Create Component → Rename "Btn/Primary"
06 Right-Click Component → "Add Variant" → Figma បង្កើត Variant Group ។ Rename Variant 1 = "Default"
07 ចុច "+" Add Variant ថ្មី → Rename = "Hover" → ប្ដូរ Fill: #1C0F07
08 Add Variant "Disabled" → Fill: #3B2007 · Opacity 35%
09 Duplicate Component ២ (Ctrl+D) → Rename "Btn/Accent" → Fill: #C89B3C · Text: #1C0F07
10 ចូល Page "Login Screen" → Assets Panel → Drag ⬡ Btn/Primary ទៅ Login Card → Override Text "Login to System →"
