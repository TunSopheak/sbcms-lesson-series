Section 01

Figma ជាអ្វី?

Figma ជា Cloud-based Design Tool ដែល Designer ទូទាំងពិភពលោក ប្រើ ។ Figma ប្រើសម្រាប់ Wireframes, UI Design, Prototyping, Design System — ទាំងអស់ក្នុង Browser ឬ Desktop App តែមួយ ។

✦
ហេតុអ្វីប្រើ Figma?
Figma ជាឧបករណ៍ Standard ក្នុង Industry — Google, Microsoft, Airbnb, Shopify ទាំងអស់ប្រើ Figma ។ Free Plan ផ្ដល់ 3 Projects + Unlimited Files ប្រើ Browser ។ Team Collaboration Real-time — ដូច Google Docs ប៉ុន្តែ Design ។

Figma vs Tools ផ្សេង

Tool Platform Price Best For
Figma ✦ Browser + Desktop Free UI Design, Collaboration, Prototyping
Adobe XD Windows, Mac Paid Adobe Ecosystem Users
Sketch Mac Only Paid Mac Designers
Canva Browser Free Graphics, Posters — មិនមែន UI Design

របៀប Setup Figma

1
ចូល figma.com
ចូល figma.com → ចុច "Get started for free" → Sign up ដោយ Email ឬ Google Account ។
2
ជ្រើស Plan
ជ្រើស Starter (Free) — គ្រប់គ្រាន់សម្រាប់ SBCMS Project ។ Free Plan: 3 Projects, Unlimited Files ។
3
Download Desktop App (Optional)
Figma ប្រើ Browser ក៏ល្អ ។ ប៉ុន្តែ Desktop App ប្រើ Font Local បាន, ល្អជាង Browser ។ Download ពី figma.com/downloads ។
4
បង្កើត New Design File
ក្នុង Dashboard → ចុច + New design file → Canvas ទទេ លេចឡើង → ចាប់ផ្ដើម! ។

Zone ទាំង 5 របស់ Figma Interface

① Title Bar — ខាងលើ
Figma Menu (File, Edit, View) · Tools Toolbar · Share Button · ប្ដូរ Tool ដោយ Shortcut ។
② Left Panel — Layers & Assets
Layers Tab: List of all Elements ជា Tree ។ Assets Tab: Components, Icons, Colors saved ។

Section 03

Frames & Pages — ស្គាល់ Frame

Frame ជា Container ដំបូងដែល Designer បង្កើត ។ Frame = ប្រភេទ Screen ១ (ឧ. Login Screen, POS Screen) ។ Elements ទាំងអស់ Nested ក្នុង Frame ។

📱
Frame vs Rectangle
Frame ជា Container ដែល Clip Children, Auto Layout Support ។ Rectangle ជា Shape ធម្មតា — Background, Card, Button ។ ប្រើ Frame សម្រាប់ Screen/Section, Rectangle សម្រាប់ UI Shape ។

Frame Presets — ទំហំ Screen Standard

iPhone 14
390 × 844

1440 × 1024
iPad
768 × 1024

SBCMS Desktop App (WinForms) → ប្រើ Frame ទំហំ 1024 × 600 ឬ 1280 × 720 ។ Web App → ប្រើ 1440 × 900 (Desktop) + 390 × 844 (Mobile) ។

របៀបបង្កើត Frame

1
ចុច F (Shortcut) ឬ ជ្រើស Frame Tool
ចុច F លើ Keyboard → Cursor ប្ដូរជា Cross → Drag លើ Canvas ។
2
ឬ ជ្រើស Preset ពី Right Panel
ចុច Frame Tool → Right Panel បង្ហាញ Presets: Phone, Tablet, Desktop → Click ១ ដង → Frame លេចភ្លាម ។
3
ដំឡើង Fill Color
Click Frame → Right Panel → Fill → ចុច Color Box → ដំឡើង Hex Code (ឧ. #FDF6EC Cream) ។
4
Rename Frame
Double-click Frame Name ក្នុង Layers Panel → វាយ ឧ. "Login Screen" → Enter ។ Name ល្អ = Layers Panel ច្បាស់ ។
Section 04

Basic Tools — ឧបករណ៍ជាមូលដ្ឋាន

Figma Toolbar (ខាងលើ Canvas) មាន Tools សំខាន់ ។ ឧបករណ៍ Figma ភាគច្រើន Shortcut ១ Key — ងាយ លឿន ។

V
Move Tool
Select, Move, Resize Elements ។ Tool Default ។ ចុច V ត្រឡប់មកវិញបានជានិច្ច ។
Shortcut: V
K
Scale Tool
Scale Element ដោយ Proportional ។ ផ្ទុយ Resize ដែល Stretch, Scale ធ្វើ Proportional ។
Shortcut: K
F
Frame Tool
បង្កើត Frame (Screen Container) ។ ជ្រើស Preset ឬ Drag Custom Size ។
Shortcut: F
R
Rectangle Tool
Draw Rectangle/Square ។ Hold Shift ពេល Drag = Square ។ ប្រើ Button, Card, Background ។
Shortcut: R
O
Ellipse Tool
Draw Circle/Oval ។ Hold Shift = Perfect Circle ។ ប្រើ Avatar, Icon Background ។
Shortcut: O
T
Text Tool
Add Text ។ Click once = Auto-width text ។ Drag = Fixed-width Text Box ។ Double-click Text = Edit ។
Shortcut: T
P
Pen Tool
Draw Custom Vector Path ។ ប្រើ Icon, Custom Shape ។ Advanced Tool — Practice Part 4 ។
Shortcut: P
H
Hand Tool
Pan/Move Canvas ដោយ Drag ។ Hold Space ពី Tool ណាក៏ Pan បាន — ត្រឡប់ Tool ដោយ Release Space ។
Shortcut: H / Space
⚡
Pro Tips — ចំណេញពេល
Ctrl+D = Duplicate Element ។ Ctrl+G = Group Selection ។ Ctrl+Z = Undo ។ Ctrl+Shift+H / V = Align Horizontally/Vertically ។ Alt+Drag = Duplicate + Move ។ Shift+Click = Multi-Select ។

Section 09

Practice — SBCMS Login Frame

ការ Practice ល្អបំផុតគឺ ធ្វើ ។ ។ ។ មិនមែនតែអាន ។ ខាងក្រោម ជា Exercise ដំបូង — Recreate SBCMS Login Screen ក្នុង Figma ។

Exercise 01 — Basic Login Frame

Task: Create SBCMS Login Screen Wireframe

01 បង្កើត New Figma File → ដំឡើង Page Name "Login Screen"
02 ចុច F → ជ្រើស Desktop Preset (1440×1024) → Rename Frame "Login Screen"
03 Fill Frame Background → Color: #1C0F07 (Espresso Dark)
04 ចុច R → Draw Rectangle 400×500 (Login Card) → Fill: #FFF9F2 → Corner Radius: 16 → Center ក្នុង Frame
05 ចុច O + Shift → Draw Circle 60×60 → Fill: #3B2007 → ដំឡើង Center Top of Card
06 ចុច T → វាយ "Sharp Bean Café" → Font: Kantumruy Pro · 24px · Bold · Color: #3B2007
07 Draw Input Fields (R) → 2 Rectangles: W=340, H=44 → Fill: White → Stroke: #EEE0CC · 1.5px → Corner: 8
08 Draw Login Button → W=340, H=48 → Fill: #3B2007 → Corner: 8 → Add Text "Login to System →" White Bold
09 Group (Ctrl+G) Login Card Elements → Rename Groups: "Logo Group", "Input/Username", "Input/Password", "Btn/Login"
10 Save (Ctrl+S) → បង្ហាញ Teacher → ✅ Exercise 01 Done!
